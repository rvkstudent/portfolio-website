Справка по работе с изображениями в Markdown

Загрузка изображения: Нажмите кнопку "Загрузить изображение" над редактором.

Markdown синтаксис для изображений:

![Альтернативный текст](путь/к/изображению.jpg "Подсказка при наведении")

Пример:

![Логотип Django](https://static.djangoproject.com/img/logos/django-logo-positive.png "Django логотип")

Подпись к изображению: Добавьте текст в курсиве сразу после изображения
